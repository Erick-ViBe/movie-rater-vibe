## Setup
1. Creating a virtual environment:\
`python3 -m venv env`

2. Activating a virtual environment:\
`source env/bin/activate`

3. Install all dependencies:\
`pip install -r requirements.txt`

4. Apply migrations:\
`python3 manage.py migrate`

## Run Project
`python3 manage.py runserver`
